# 트럭

Solved at: 2026-04-11 (1h)

Offline Query + Greedy + Binary Search

<details>
  <summary><strong>문제</strong></summary>

회사는 하나의 트럭을 이용해 물건을 운송하려 한다. 트럭의 크기는 정수로 표현되며, 크기가 클수록 더 많은 물건을 실을 수 있다.

총 $N$명의 소비자가 있고, 각 소비자 $i$는 여러 개의 제안을 가지고 있다. 각 제안은 두 정수 $(S_{i,j}, P_{i,j})$로 주어지며, 다음을 의미한다.

- $S_{i,j}$: 이 제안을 사용하기 위한 최소 트럭 크기
- $P_{i,j}$: 해당 조건에서 소비자가 지불할 수 있는 금액

트럭의 크기를 $sz$로 정했을 때, 각 소비자는 다음과 같이 행동한다.

- $S_{i,j} \le sz$를 만족하는 제안들 중에서 가장 큰 $P_{i,j}$를 선택하여 지불한다.
- 조건을 만족하는 제안이 하나도 없다면, 해당 소비자는 아무 것도 지불하지 않는다.

회사의 총 매출은 모든 소비자가 지불한 금액의 합이다.

$$
f(sz) = \sum_{i=1}^{N} (\text{consumer } i \text{가 선택한 최대 } P_{i,j})
$$

이제 $Q$개의 시나리오가 주어진다. 각 시나리오마다 목표 매출 $Q_k$가 주어질 때,

- 총 매출이 $Q_k$ 이상이 되도록 하는 최소 트럭 크기 $sz$를 구하라.
- 그러한 트럭 크기가 존재하지 않으면 $-1$을 출력하라.

<summary><strong>입력</strong></summary>

첫째 줄에 소비자의 수 $N$이 주어진다.

다음 $N$개의 블록에 대해:

- 각 소비자마다 제안의 개수 $c_i$가 주어진다.
- 이어서 $c_i$개의 줄에 걸쳐 $(S_{i,j}, P_{i,j})$가 주어진다.

그 다음 줄에 질의의 개수 $Q$가 주어진다.

다음 줄에 $Q$개의 목표 매출 $Q_k$가 주어진다.

- $1 \le N \le 10^5$
- $\sum c_i$는 최대 $10^5$ 수준
- $1 \le S_{i,j} \le 10^9$
- $1 \le P_{i,j} \le 10^9$
- $1 \le Q \le 10^5$

<summary><strong>출력</strong></summary>

각 질의에 대해, 조건을 만족하는 최소 트럭 크기를 공백으로 구분하여 출력한다.

- 존재하지 않으면 $-1$을 출력한다.
